Module set

A set is a compact, immutable set of strings, queried by binary search.

It stores many short strings as a sorted blob of key bytes plus a uint32 offset array, with no per-key allocation, so it costs about length + 4 bytes per key, instead of the hash node and slab rounding a Lua or rcu table pays per key. Built once by set.new (which sorts the members), it is then read-only: lookups take no lock and allocate nothing, safe from softirq. It tests exact membership (set:has) or matches by segment, from the right (set:suffix, as for domains) or the left (set:prefix, as for filesystem or URI paths).

See also:

    rcu

Class set

A built set. Query it with set:has for exact membership, set:suffix or set:prefix for segment membership, and # for size.
set:__len ()
Returns the number of keys in the set.

Returns:

    integer the number of keys.
set:has (s)
Returns whether a string is in the set.

Parameters:

  • s string the string to test.

Returns:

    boolean whether s is in the set.
set:prefix (s, sep)
Returns whether s, or a prefix of it before a separator, is in the set.

Tests s, then each prefix ending before a sep, so t:prefix("/a/b/c", "/") tries "/a/b/c", "/a/b", "/a". This roots the hierarchy on the left, as for filesystem or URI paths.

Parameters:

  • s string the string to test.
  • sep string a one-byte separator.

Returns:

    boolean whether s, or a prefix of it, is in the set.
set:suffix (s, sep)
Returns whether s, or a suffix of it after a separator, is in the set.

Tests s, then each suffix starting after a sep, so t:suffix("a.b.com", ".") tries "a.b.com", "b.com", "com". This roots the hierarchy on the right, as for domains, reverse-DNS, or dotted ids.

Parameters:

  • s string the string to test.
  • sep string a one-byte separator.

Returns:

    boolean whether s, or a suffix of it, is in the set.

set

new (strings)
Builds the set from an array of strings.

The array is sorted in place. The members must be unique; duplicates are kept, which keeps lookups correct but wastes space, so de-duplicate first if it matters. It allocates, so call it where it can sleep (a process-context runtime, or any runtime's load).

Parameters:

  • strings {string,...} the member strings, in any order.

Returns:

    set the built set.

Raises:

Error on a non-string member, if the keys exceed 4 GiB, or on allocation failure.

Usage:

    local blocked = set.new({ "evil.com", "ads.net" })
