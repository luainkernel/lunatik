Module cpu

Linux CPU Lua interface.

This module provides access to Linux's CPU abstractions.

stats (cpu)
Gets CPU statistics for a specific CPU. Fetches kernel CPU statistics including user, nice, system, idle, iowait, irq, softirq, steal, guest, guest_nice and forceidle times.

Parameters:

  • cpu integer The CPU number (0-based) to query.

Returns:

    table A table containing CPU time statistics with the following fields:

Raises:

Error if CPU is offline

Usage:

    local cpu0 = cpu.stats(0)
print("CPU 0 user time:", cpu0.user)
print("CPU 0 idle time:", cpu0.idle)
