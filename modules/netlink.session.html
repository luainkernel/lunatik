Module netlink.session

Netlink request/response session.

A session owns an AF_NETLINK socket and the transaction discipline on it: talk and dump send one request and drain its complete reply, raising on a kernel error reply (NLMSG_ERROR). Protocol modules (rtnetlink, generic netlink) derive from this class. All methods block and require a sleepable runtime.

See also:

    socket

session:close () Closes the underlying socket.
