Class netfilter_hook

Registered Netfilter hook. Garbage collecting this object unregisters the hook.
netfilter_hook.action
Table of Netfilter hook verdicts (actions). These determine the fate of a packet processed by a hook. NF_* come from <linux/netfilter.h>, XT_* from <linux/netfilter/x_tables.h>; the two prefixes are mixed, so this table is kept here rather than autogenerated.

Fields:

  • DROP integer Drop the packet silently.
  • ACCEPT integer Let the packet pass.
  • STOLEN integer Packet is consumed by the hook; processing stops.
  • QUEUE integer Queue the packet to a userspace program.
  • REPEAT integer Re-inject the packet into the current hook (use with caution).
  • STOP integer Terminate rule traversal in the current chain (iptables specific).
  • CONTINUE integer Alias for ACCEPT, primarily for Xtables.
  • RETURN integer Return from the current chain to the calling chain (iptables specific).
netfilter_hook.bridge_hooks
Table of Netfilter hooks in the BRIDGE family. Kept in C because NF_BR_* and NF_BR_PRI_* share the same prefix; the priorities are exported separately via require("linux.nf").br.pri.

Fields:

  • PRE_ROUTING integer Before any bridge processing.
  • LOCAL_IN integer Destined for the bridge interface itself.
  • FORWARD integer Forwarded between bridge ports.
  • LOCAL_OUT integer Originating from the bridge interface.
  • POST_ROUTING integer After bridge processing, before egress.
netfilter_hook:register (opts)
Registers a Netfilter hook.

Parameters:

  • opts table Hook options: hook (function), pf, hooknum, priority (integers), and optionally mark (integer, default 0).

Returns:

    netfilter_hook Registered hook handle.
