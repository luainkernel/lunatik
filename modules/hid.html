Class hid_driver

Represents a registered HID driver.
hid_driver:register (opts)
Registers a new HID driver.

Parameters:

  • opts table driver options: name (string), id_table (array of device ID tables, each with optional integer fields bus, group, vendor, product, driver_data)

Returns:

    hid_driver

Raises:

if required fields are missing, id_table is invalid, or driver registration fails
